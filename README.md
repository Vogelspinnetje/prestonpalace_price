# Preston Palace Price
## Description
Preston Palace prices can change from time to time. This script retrieves the prices for 1 night for 1 person over a user-defined time period. If the price matches the user-specified price_cap, the corresponding day is added to the iCalendar.

## Usage
- Install the necessary requirements.
- Create a file called "ics_path.txt" which should contain the path to the .ics file
- Run price_to_agenda.py to update your .ics calendar with the new events
- You can change the timespan and price_cap on line 22 of price_to_agenda.py
